Complete (or near-complete) Verilog solutions to all problems on **[HDLBits](https://hdlbits.01xz.net/wiki/Main_Page)** — the best online platform for practicing digital hardware design using Verilog HDL.

This repository is created for learners, students, and FPGA/VLSI enthusiasts who want clean, well-commented reference solutions while learning Verilog step-by-step.

**Warning**: Please attempt the problems yourself first!  
Use this repo only when you're stuck or for verification. Copy-pasting defeats the purpose of learning.

## Why This Repository?

- Clean, minimal, and efficient Verilog code
- Proper comments explaining the logic (great for beginners)
- Organized folder structure matching HDLBits sections exactly
- 100% passing solutions (tested on HDLBits as of February 2026)
- Regularly updated if new problems are added


Each `.v` file is named exactly as the problem (e.g., `mt2015_q4.v`, `exam_a1.v`, etc.)

## How to Use

1. Go to [HDLBits](https://hdlbits.01xz.net)
2. Solve problems in order (recommended path: Getting Started → Verilog Language → Circuits)
3. If stuck → come here → search for the problem name → read the commented solution
4. Understand the logic → close the file → write it yourself again
